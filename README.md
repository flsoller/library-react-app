## A project to keep track of and manage your book library.

Planned stack:
Frontend: ReactJS (https://reactjs.org)
UI: Material-UI (https://material-ui.com/)

## Planned features:

Home page with selection to view library by:

- All books
- Loaned out
- Reading status
- Favorites
